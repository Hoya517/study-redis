# Practice 2 - Redis Only Data Storage

## 📌 목표

Spring Entity 없이 Redis에 직접 데이터를 저장하는 실습

------------------------------------------------------------------------

## 🛠 Tech Stack

-   Java 17
-   Spring Boot 3.x
-   Spring Data Redis
-   RedisTemplate
-   Redis (Docker)

------------------------------------------------------------------------

## 📂 주요 구현 내용

### 1️⃣ Entity 없이 Redis 저장

-   실제 DB Entity 생성하지 않음
-   Redis에 직접 데이터 저장
-   RedisTemplate 활용

------------------------------------------------------------------------

### 2️⃣ RedisTemplate 사용

-   opsForValue()
-   opsForHash()
-   직접 Key 설계
-   자료형 직접 제어

------------------------------------------------------------------------

## 📌 학습 포인트

-   Repository 없이 Redis 활용 방법
-   RedisTemplate 구조 이해
-   직렬화 / 역직렬화 동작 방식 이해
-   자료형 직접 제어의 장단점

------------------------------------------------------------------------

## 📚 요약

RedisTemplate을 사용하면 자료형을 직접 선택하고 제어할 수 있지만
코드량은 증가한다.
